基本介紹

原本是 3D 世界的主角方塊，因為不小心將世界的方塊給擊碎了，從而迷失在 2D 的空間中。為了尋回這世界原本的樣貌，開始踏上蒐集 3D 碎塊的迷途之旅。


Pixel Maze 使用 3D 引擎架構出三維空間世界，再以 2D 的方式呈現遊戲體驗，讓玩家能徘徊在不同象限的視角，用特殊的 2D 的角度暢遊與探索我們所熟知的三維世界。


遊戲的方式類似一般的 Plat-form 遊戲，但別於傳統的是，主角沒有跳躍的能力。必須透過技巧性的操作才能穿越障礙。


遊戲的風格結合了復古的 Pixel 與現代簡約的幾何風格構出整個世界，搭配復古遊戲曲風 ( Chiptune ) 的音樂，呈現出復古又不失科技感的奇幻氛圍。
